Create

With Ephemeral, posting an event is faster and easier than ever. Choose a time and place, add a title, and start sharing. You can also add photos, and detailed description, and more.

Find exciting events at a glance with Ephemeral's inutitive map-based interface. The data-rich display makes it easy to identify event categories, popularity, distance, and immediacy. See events from many sources, including Eventbrite, Eventful, and Ephemeral's userbase, all in the same place.

About Ephemeral

Inspiration

We wanted to create an app that answers the question, "What's happening near me right now?" We wanted to make it easy for people to find new things in their communities, and easy for travelers to quickly plug in to a new place.

Performance

Ephemeral's stack is designed from the ground up for speed and scalability. We use a heavily indexed PostgreSQL database, multiple Node.js servers, and various services to keep server response time as fast as possible. In the background, a separate server is gathering and parsing event information from all over the web. The client application runs on Ionic and Angular.JS, provide tons of responsiveness, and instantaneous filtering and search.

Design

How do you minimize the number of clicks it takes a user to find an appealing event? We did it by making our default map view information-rich without being overwhelming, and by putting an excellent set of filters at the user's fingertips.
